method1 Using Cyclic Replacements

  • We need to be aware that if n%k=0 there might be more than one cycle. However, we can prove that if at some point we hit the original position with some elements not being moved, then the next position is guaranteed not moved.
  • Therefore we can keep a count variable to keep track of the number of elements that have been moved.

method2 reversion

  • First reverse the entire string
  • Then reverse the first k elements
  • At last reverse the left elements
