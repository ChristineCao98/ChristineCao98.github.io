134. Gas Station

Posted on 2021-02-27 Edited on 2021-02-28 In algorithm
  • It’s impossible to perform the road trip if sum(gas) < sum(cost). In this situation the answer is -1.
  • It’s impossible to start at a station i if gas[i] - cost[i] < 0, because then there is not enough gas in the tank to travel to i + 1 station.
  • Algorithm directly ensures that it’s possible to go from N to the station 0. And by using proof of contradiction we can also prove that it is possible for the last part of the round trip from the station 0 to the station N.
