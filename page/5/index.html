parameterType

  • 如果参数是简单类型(八个基本类型＋String)，则可以使用任何占位符，#{xxxx}(也可以用${value},但占位符只能是’value’)
  • 如果是对象类型，则必须是对象的属性，#{属性名}(或${属性名})，严格区分大小写
  • #{}和&{}
    • 相同
      • 都可以获取对象的值(嵌套类型对象)
        • 模糊查询
          • 方式一：person.setName(“%ch%”)(用#{name})
          • 方式二：’%${name}%’
        • 输入对象为HashMap
          example: where id=#{personid}
          用map中key的值匹配占位符#{personid}，如果匹配成功，就用value值替换它
    • 不同
      #{}自动给String加上引号，${}原样输出，但是适合用于动态排序
      example:select name,age,personsex,id from person order by ${value} desc
  • #{}可以防止sql注入，${}不行
    • 调用存储过程实现查询，statementType=CALLABLE
      存储过程的输入参数，在MyBatis用map传递(HashMap),输出参数通过map的get方法获得

Basic

  • Two architectures
    • CS architecture
      client server
    • drawbacks
      • If software needs updating, all of them should be updated
      • Hard to maintain: you need to maintain every client software
      • Every client needs to install client software
    • BS architecture
      • browser server
      • Client can visit server through browsers
  • Common status code
    • 200: successful
    • 300/301: redirection
    • 404: resources do not exist
    • 403: lack authorization
    • 500: server error(usually it is because of false code)
  • We can specify the default page by editing in project/WEB_INFO/web.xml.
  • virtual path
  • method 1(need rebooting)
    To configure web project to directories other than webapps.
    In host tag of conf/server.xml:
    • docBase=”” actual path
    • path=”” virtual path(absolute path/relative path(relative to webapps))
      It means that visiting path equals visiting docBase
  • method 2(doesn’t need rebooting)
    Create a new file named project_name.xml in D:\xampp\tomcat\conf\Catalina\localhost and add a row which is the same as method 1.
  • virtual host
    • the process of DNS

      If we wanna visit a domain, we will first look for local resolution. If there is not a mapping, we will then look for internet resolution.
    • method
      For example if we wanna visit localhost by www.test.com
      • In conf/server.xml:
        <Host appBase=””(the actual path of your project) name=”www.test.com">
        <Context docBase=””(actual path) path=”/“(root directory)>
      • Change default host to www.test.com
      • Add a ip-domain name mapping to host file in C:\Windows\System32\drivers\etc

isPowerOfThree

Posted on 2021-02-11 In algorithm

Knowing the limitation of n, we can now deduce that the maximum value of n that is also a power of three is 1162261467.
Therefore, the possible values of n where we should return true are 3^0, 3^1… 3^19. Since 3 is a prime number, the only divisors of 3^19 are 3^0, 3^1 … 3^19, therefore all we need to do is divide 3^19 by n. A remainder of 0 means n is a divisor of 3^19 and therefore a power of three.
